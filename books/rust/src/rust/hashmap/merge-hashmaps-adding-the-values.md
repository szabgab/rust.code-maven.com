# Merge two HashMaps adding the values

* HashMap

Given

```
{
    "a" : 1,
    "b" : 2,
}

{
    "a" : 3,
    "c" : 5,
}
```

Adding them together should result in

```
{
    "a" : 4,
    "b" : 2,
    "c" : 5,
}
```
