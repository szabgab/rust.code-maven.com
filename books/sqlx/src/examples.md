# Source code and examples

The source code of SQLx on [GitHub](https://github.com/launchbadge/sqlx/)

Clone the repository

```shell
$ git clone https://github.com/launchbadge/sqlx/
```

Install the `sqlx` that is distributed as [sqlx-cli](https://crates.io/crates/sqlx-cli) by

```shell
$ cargo install sqlx-cli
```

The `todos` examles have README files.
