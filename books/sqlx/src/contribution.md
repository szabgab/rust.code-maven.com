# Tests and contribution

The source code of SQLx on [GitHub](https://github.com/launchbadge/sqlx/)

Clone the repository

```shell
$ git clone https://github.com/launchbadge/sqlx/
```

To run the tests that come with the source code read the [README in tests folder](https://github.com/launchbadge/sqlx/tree/main/tests/).
The following command will use a lot of memory and CPU:


```shell
$ cd tests
$ docker compose up -d
$ ./x.py --clippy
```

See also [CONTRIBUTING.md](https://github.com/launchbadge/sqlx/blob/main/CONTRIBUTING.md) in the repository.
